# Social Media — Pika's Arbeitsbereich

## Ordnerstruktur
- **drafts/** — Entwürfe (Captions, Beschreibungen, Threads) zur Freigabe durch Paul
- **calendar/** — Posting-Kalender und Planungen
- **templates/** — Wiederverwendbare Vorlagen (Caption-Strukturen, Hashtag-Sets, CTA-Bausteine)
- **assets/** — Thumbnail-Briefings, Bild-Referenzen, Brand-Guidelines

## Workflow
1. Paul oder Turtok legen Aufgabe an (z.B. "5 Reels für diese Woche planen")
2. Pika erstellt Drafts in `drafts/`
3. Paul gibt Freigabe
4. Pika plant Publishing

## Regeln
- Nichts wird ohne Pauls Freigabe veröffentlicht
- Jeder Draft hat: Hook, Value, CTA
- Dateinamen: `YYYY-MM-DD-plattform-thema.md`
